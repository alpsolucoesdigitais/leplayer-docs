Conhecendo o Dashboard

Conhecendo o Dashboard

O dashboard do LePlayer foi projetado para ser extremamente simples.

Vamos conhecer cada elemento.

1. Barra de navegação / Menu superior

Aqui temos as opções para que o usuário transite dentro da plataforma:

  • Players (Acesso ao dashboard)
  • Conta (Acesso a informações pessoais, do negócio, configurações de segurança e exclusão de conta)
  • Pagamento (Visualização de status da assinatura e acesso à plataforma de pagamentos)
  • Suporte (Acesso à documentação completa, chat ao vivo com a equipe de suporte, contato por e-mail e perguntas frequentes / FAQ)
  • Sair (Botão de sair da plataforma)

2. Total de Players

Aqui são exibidos quantos players o usuário possui no total.

3. Botão Criar Novo Player

Este botão dá acesso à página de criação de um novo player. 

4. Barra de pesquisa

Permite a busca pelo nome do player.

5. Ordenação de players

Permite o ordenação por mais recentes, mais antigos e ordem alfabética crescente ou decrescente.

6. Grade de players

Aqui há a visualização dos players, incluindo título, data de criação, minitatura e 3 botões.

  • Editar (abre a edição do player selecionado)
  • Embed (Permite a incorporação rápida de um player copiando o código de incorporação para a área de transferência
  • URL (copia a URL / link do player para a área de transferência. Ideal para incorporação em ferramentas que dão opção de embed/incorporação via  URL)

7. Barra de paginação

Aqui o usuário pode navegar entre as páginas onde estão seus diversos players. Para maior desempenho, apenas 12 players podem ser visualizados por página.
