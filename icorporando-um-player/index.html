Incorporando um Player

Incorporando um player por código de incorporação

Para incorporar um player no seu site ou aplicação, basta copiar o código de incorporação e colar no local desejado.

Opções para obter o código:

No dashboard abaixo da miniatura do player através do botão Embed.

Nas páginas "Criar Novo Player" e "Editar Player" através do botão Copiar Código de Incorporação.

Em seguida, cole esse código no seu site / aplicação. O funcionamento deve ser imediato.

 

IMPORTANTE: PARA QUE FUNCIONE CORRETAMENTE EM APLICAÇÕES NÃO WEB, DEVE HAVER COMPATIBILIDADE COM HTML, CSS E JAVASCRIPT. FALE A RESPEITO COM SEU DESENVOLVEDOR.

 

Incorporando um player por link de incorporação

Certas plataformas possuem sistema próprio de incoração através de um link / URL. Isso permite maior compatibilidade, pois o iframe já vem ajustado de acordo com a mesma. Clique aqui para saber o que é um iframe.

Opções para obter o link:

 

No dashboard abaixo da miniatura do player através do botão URL.

Nas páginas "Criar Novo Player" e "Editar Player" através do botão Copiar URL de Incorporação.

Cole esse link no campo apropriado seu site / aplicação. O funcionamento deve ser imediato.
